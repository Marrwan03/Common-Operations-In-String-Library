Common Operations In String Library

Using MyName: [ Marwan ]
Function Describe Process Result
Contains(char value); To Check if the Name contains the character or not. MyName.Contains('j'); False
ToLower(); To Change all letters to lowercase. MyName.ToLower(); marwan
ToUpper(); To Change all letters to uppercase. MyName.ToUpper(); MARWAN
Insert(int startIndex, string value); To Insert the value into your name. MyName.Insert(0, "Mr. "); Mr. Marwan
Remove(int startIndex); To Remove from start index until the end of the name. MyName.Remove(3); Mar
Replace(string oldValue, string newValue); To Replace old value with new value. MyName.Replace("wan", "tin"); Martin
Substring(int startIndex); To Get a substring starting from specified index. MyName.Substring(2); rwan
Substring(int startIndex, int length); To Get a substring with specified starting index and length. MyName.Substring(1, 3); arw
StartsWith(string value); To Check if the name starts with specified value. MyName.StartsWith("Mar"); True
EndsWith(string value); To Check if the name ends with specified value. MyName.EndsWith("wan"); True
IndexOf(char value); To Get the index of first occurrence of specified character. MyName.IndexOf('w'); 3
Length To Get the length of the name. MyName.Length 6
Trim(); To Remove all leading and trailing whitespace. " Marwan ".Trim(); Marwan
Split(char separator); To Split the string into an array based on separator. "Mar-wan".Split('-'); ["Mar", "wan"]
Equals(string value); To Check if the name equals specified value. MyName.Equals("marwan"); False
ToCharArray(); To Convert the name to a character array. MyName.ToCharArray(); ['M','a','r','w','a','n']
